# Lua

Curso de programación en Lua, un lenguaje de programación ligero, rápido y fácil de aprender, ampliamente utilizado en el desarrollo de videojuegos, aplicaciones embebidas y scripting en general.

En este repositorio, encontrarás una serie de apuntes y recursos para aprender Lua, incluyendo tutoriales, ejemplos de código y enlaces a documentación adicional. El objetivo es proporcionar una guía completa para aquellos que quieran aprender Lua desde cero o mejorar sus habilidades en este lenguaje.

Esta página esta realizada con `mkdocs`, una herramienta de documentación estática que facilita la creación de sitios web para proyectos de software. El contenido se organiza en diferentes secciones, cada una dedicada a un aspecto específico del lenguaje Lua, como variables, operadores, estructuras de control, funciones, módulos y más.

Puedes ejecutar el sitio web localmente utilizando el comando `mkdocs serve` en la terminal, lo que te permitirá navegar por los apuntes de manera interactiva. Además, el sitio está diseñado para ser fácil de leer y navegar, con enlaces a recursos adicionales para profundizar en cada tema.

Esperamos que este curso de programación en Lua sea útil para aquellos que quieran aprender este lenguaje de programación versátil y poderoso. ¡Feliz aprendizaje!
