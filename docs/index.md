# Apuntes de Programación en Lua

En la presente página se recopila información sobre el lenguaje de programación Lua, incluyendo tutoriales, ejemplos y recursos útiles para aprender y dominar este lenguaje. Lua es un lenguaje de programación ligero, rápido y fácil de aprender, ampliamente utilizado en el desarrollo de videojuegos, aplicaciones embebidas y scripting en general.
